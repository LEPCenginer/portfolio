# Portfolio

Este proyecto ahora funciona como un sitio web estático de una sola página, construido con HTML, CSS y JavaScript.

## Desarrollo local

Puedes abrir directamente el archivo [src/index.html](src/index.html) en el navegador o ejecutar:

```bash
npm start
```

Esto levantará un servidor local en http://localhost:8000/.

## Estructura principal

- [src/index.html](src/index.html): contenido del portafolio.
- [src/styles.css](src/styles.css): estilos visuales.
- [src/script.js](src/script.js): pequeñas mejoras interactivas.
